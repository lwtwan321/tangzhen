# tangzhen
2月份公司裁员，一下子工作了10年 想撑这个机会家里休息一段时间，顺便自己写个简单的android MVP框架。

整个项目思路，以前两个小公司开发都是一个module打天下，但现在流行细分，每个大模块分每个模块，所以在家休息的这段时间里自己写个细分模块的框架。
